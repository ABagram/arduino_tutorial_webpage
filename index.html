Procedures

Connecting the Board to the Development Environment

  1. Plug the Arduino Uno board to one of the USB ports of the laptop/ computer through the USB Type-A to USB Type-B cable. This should light up the LED marked "ON" on the Arduino Uno to indicate that the board is now being powered.
  2. Launch the Arduino IDE.
     Warning! Laptops/ computers generally can recognize that there is a USB device connected, but are not able to automatically identify it as an Arduino board.
    1. Tools  Board, then select Arduino Uno.
    2. Tools  Port, then select the communication (COM) port that corresponds to the Arduino.
      3. info Note. The port can be identified through the system’s Device Manager, wherein the Arduino Uno appears as an “Unknown device”.

Connecting the Breadboard to the Arduino

Some breadboards do not have markings that visualize the orientation of the power rail, which is typically marked by a red line and a plus () sign, and the ground rail, which is typically marked by either a blue or black line and a negative sign. In such case, simply assign the rails. Generally, the left rails are used as the power rails while right rails are used as ground rails. Make sure to keep it consistent to prevent accidental short circuits.
  1. To be able to use both power rails as one, establish the connection between the two power rails of the breadboard by connecting one hole from each rail with a M/M jumper wire.
    Tip: Use a red jumper wire to indiicate positive power supply connection for consistency.
  2. Connect one of the holes witin the power rail to the 5V pin of the Arduino Uno using a M/M jumper wire.
